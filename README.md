# ABCT Technology Committee Website

Website for the Association of Behavioral and Cognitive Therapies (ABCT) Technology Committee.

## Features

- **Home Page**: Welcome section with overview of committee features
- **Updates Section**: Post and view latest updates and announcements
- **File Repository**: Upload and manage shared files and documents
- **Private Collaboration Area**: Secure section for committee members (password: `abct2026`)
- **Secret Cat Mode**: Hidden Easter egg feature - press the keys `c`, `a`, `t` in sequence to activate a walking cat animation

## Usage

Simply open `index.html` in a web browser. No server or build process required.

### Navigation

- Use the navigation menu to switch between sections
- All data is stored locally in the browser's localStorage

### Private Section

Access the private collaboration area with the password: `abct2026`
